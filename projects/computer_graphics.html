Computer Graphics

2020

A game developed with javascript to improve the knowledge of computer graphics.

Contains a user interface to control some features, such as speed, scale, textures and camera view

Controls:

  • W - Accelarate
  • S - Brake
  • A - Turn left
  • D - Turn right
  • P - Autopilot
  • L - Lauch the package
  • R - Reset scene

For more info please check on Github or click on the Link for the game.
