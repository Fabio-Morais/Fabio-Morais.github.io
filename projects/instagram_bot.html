Instagram Bot

2020

The main goal of this project was to learn about selenium and automation tools with Python. The main concept behind this Instagram bot was to create a bot that could automatically participate in giveaways.

Features:

  • Automatically commenting photos and tag people;
  • Following;
  • Liking photos;
  • Get a list of users and save as .txt;
  • Automatically login into the account;
